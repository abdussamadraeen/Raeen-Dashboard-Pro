How to Install

Setting up Raeen Dashboard Pro takes less than 60 seconds. Follow these steps to get started with your new digital workspace.

1

Download the ZIP

Download the latest release ZIP file from our GitHub repository. This contains all the source code, icons, and assets required for the extension.

Download v1.0.0 ZIP
2

Extract the Files

Extract the contents of the ZIP file to a folder on your computer. We recommend keeping it in a dedicated "Extensions" folder where it won't be moved.

3

Open Extension Manager

Open Google Chrome or Microsoft Edge and navigate to chrome://extensions (or edge://extensions). Enable Developer Mode in the top right corner.

4

Load Unpacked

Click the Load unpacked button and select the folder you extracted in Step 2. The Raeen Dashboard will appear in your list of extensions.

5

Enjoy!

Open a new tab and enjoy your new premium dashboard. Don't forget to pin the extension for quick access to settings!
